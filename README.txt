File structure of the SoccerWorld website

Root dir: contains all the main page tabs of the website, such as the homepage (home.html), about, etc.

css: contains the css files that our webpages use

databasefiles: contains the sql files that can be used to build the database

html: contains the page templates (headers and footers) used by our php scripts to build our web pages dynamically, for example, each new product page starts with the productupper.html, then our php scrip fills in the body, and it's ended with the productlower.html template

phpmethods: all of our php scripts are kept here

productimages: all of the product images are stored here

products: all of our product page php files are stored here, in the form productid.php
